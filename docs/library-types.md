# Introduction
The platform support many different types of libraries.

# Library Types
## Simple Library
This is a traditional library, that has one or more physical locations where goods are borrowed and returned by borrowers
It will usually have a person there acting to create Loans, and accept the returns.  The goods are legally the property of
the library itself.

## Distributed Library
A distributed library is a collection of users, who all agree to borrow and lend goods to one another.  Pickup and return
must be coordinated by the borrow and lender.  The library itself does not "own" the goods.  On return, both the borrower
and lender must mark the item as returned to limit any opportunity for fraud, and in addition location services or photos
may be used to provide evidence of the return.

Technology like automatic lockers may also be used to make pickup and return easier.

## Free Store
A free store is similar to a Simple Library, but all items there don't have a return date mandated.
Users can donate and take goods here.  Technology such as NFC or RFID chips can be used to allow automatic inventory.

## BuyNothing Groups
BuyNothing groups exist in many forms of social media.  Adapters to this allow us to treat them as a form of library as well,
where any goods are lent without a return date, and similar to the DistributedLibrary pickup and return must be coordinated between buyers.
