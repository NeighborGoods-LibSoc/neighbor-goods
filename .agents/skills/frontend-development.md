# Skill: Frontend Development (Next.js & Tailwind)

This skill focuses on building the NeighborGoods user interface with a "warm and neighborly" vibe.

## Tech Stack
- **Framework**: Next.js (App Router).
- **Styling**: Tailwind CSS.
- **Components**: Radix UI / Shadcn UI (where applicable).

## Vibe & UI/UX
- **Tone**: Friendly, casual, non-corporate.
- **Visuals**: Soft edges, warm colors (F0EAD2, DDE5B6, ADC178), hand-drawn or friendly icons.
- **Responsive**: Mobile-first approach.

## Best Practices
- **Component Reusability**: Place shared UI components in `src/components/`.
- **Server Components**: Prefer React Server Components (RSC) for data fetching where possible.
- **Accessibility**: Ensure high contrast for readability and proper ARIA labels.
